# MyReads Project

My Reads Project, React application

## Getting Started
Run:
```
npm install
```
and then
```
npm start
```
now open your browser on [`localhost:3000`](#localhost:3000)

## This Project using:
* [`NodeJS`](https://nodejs.org/en/)
* [`NPM`](https://www.npmjs.com/)
* [`ReactJS`](https://reactjs.org/)
